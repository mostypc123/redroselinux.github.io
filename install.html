Installation Guide

To install Redrose Linux, the first step is to download the disk image. The image size is approximately 1.5-2 GB, which is smaller than most standard distributions.

After downloading, you need to flash the disk image onto a USB drive or another removable drive. Connect the drive to your current device before proceeding.

On Windows, download Rufus and follow the instructions. Simply select the downloaded disk image and your removable drive, then start the flashing process.

On Linux, run the following command in a terminal: sudo dd if=~/Downloads/redroselinux.iso of=/dev/sdX bs=4M status=progress && sync Make sure to replace /dev/sdX with the correct device name for your removable drive!

Next, you need to boot from the USB drive. Reboot your computer and press one of the following keys to access the BIOS/UEFI:
Esc, Enter, F9, Del.
If a menu listing drives appears, select your removable drive. If a menu with boot options appears instead, select Boot Menu or a similar option, and then choose your USB drive.

Once booted, the installer will start. Wait a moment while the system loads—the installer will handle most tasks automatically until the setup interface appears.

Follow the on-screen instructions. Here's a quick reference for some terms you may encounter:

  • Username - your personal user account name
  • Root - the system administrator account
  • Console keymap - your keyboard layout (e.g., us, uk, sk)
  • Timezone - your timezone in Region/City format (e.g., Europe/Bratislava)
  • Locale - system language setting (e.g., en_US.UTF-8)

After completing these steps, the installer will finish the installation automatically.
